## Install

### install with curl

```bash
curl -fsSL https://raw.githubusercontent.com/PeonPing/peon-ping/main/install.sh | bash -s -- --packs=ae_qianyu
```

Then start or restart the claude.

## What you'll hear

| Event | Examples |
|---|---|
| Session starts | *"你好呀管理员，陪我练练剑吗?"* |
| Task finishes | *"这次轻松"*|
| Permission needed | *"快!用这个!"* |
| Dang! | *"当破即破"*, *"当断即断"*, *"当弃即弃"*|

## How it works

peon-ping uses hook technology to trigger sound effects on Claude's event. The response speed is much faster compared to implementing this function with skill.
